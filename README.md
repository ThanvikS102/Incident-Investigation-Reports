# Incident Investigation Reports

A collection of cybersecurity incident investigation reports developed from **CyberDefenders Blue Team Labs**. Each report follows the **NIST SP 800-61 Incident Response** methodology and demonstrates the investigation process used to analyze simulated security incidents.

The objective of this repository is to showcase practical Blue Team and SOC Analyst skills through structured investigations, technical analysis, IOC extraction, and professional incident documentation.

---

## Objectives

- Perform end-to-end incident investigations using realistic forensic artifacts.
- Practice SOC analyst workflows for identifying, analyzing, and documenting security incidents.
- Extract and document Indicators of Compromise (IOCs).
- Map attacker behavior to the MITRE ATT&CK framework.
- Produce professional incident reports following industry best practices.
