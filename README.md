# Incident Investigation Reports

## Overview

This repository contains five cybersecurity incident investigation reports created from CyberDefenders Blue Team Labs. Each investigation follows the NIST Incident Response lifecycle and documents the complete analysis process, from initial alert triage to containment and recommendations.

## Objectives

* Practice Tier 1 SOC analyst workflows
* Analyze security incidents using real forensic artifacts
* Identify Indicators of Compromise (IOCs)
* Map attacker behavior to the MITRE ATT&CK framework
* Produce professional incident reports
